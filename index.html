GIS - Lab project
by Group - 4

GIS Lab 2020 - 2021

This project was developed by Geoinformatics students of group-4. It asseses, through GIS & Statistical analysis, the accuracy & reliability of two population datasets (GHS - 2015 & Worldpop - 2015) in order to inform users on the benefits and limitations derived from their usage. It contains seven base-maps, in addition to the requested layers.

Technologies

GeoServer

GeoServer is a Java-based server that allows users to view and edit geospatial data. Using open standards set forth by the Open Geospatial Consortium (OGC), GeoServer allows for great flexibility in map creation and data sharing.

QGIS

QGIS is a free and open-source cross-platform desktop geographic information system application that supports viewing, editing, and analysis of geospatial data. All the layers were produced using QGIS.

OpenLayers

OpenLayers is an open-source JavaScript library for displaying map data in web browsers as slippy maps. It provides an API for building rich web-based geographic applications similar to Google Maps and Bing Maps.

JavaScript

JavaScript is a text-based programming language used both on the client-side and server-side that allows you to make web pages interactive.
